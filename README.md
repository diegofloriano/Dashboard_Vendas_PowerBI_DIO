# Dashboard de Vendas - Power BI (DIO)

Este é o meu primeiro projeto prático de Business Intelligence desenvolvido durante o curso da **DIO**. O objetivo foi transformar um dataset bruto de vendas em um relatório visual e interativo para facilitar a tomada de decisão.

## 📊 Visão Geral
[cite_start]O dashboard analisa o desempenho de vendas e lucro entre os anos de **2013 e 2014** [cite: 5, 45][cite_start], cobrindo diferentes produtos, segmentos e localizações geográficas[cite: 9, 42, 104].

### Principais Métricas (KPIs)
* [cite_start]**Vendas Totais:** 118,73 Mi[cite: 65].
* [cite_start]**Produto Líder:** **Paseo**, representando **29,35%** do faturamento[cite: 7, 14].
* [cite_start]**Segmento Mais Lucrativo:** **Government**, com **65,04%** do lucro total (11,39 Mi)[cite: 116, 121].

## 📈 O que o Dashboard responde?
1. [cite_start]**Performance de Produtos:** Identifica qual produto traz mais receita (Paseo) [cite: 7] [cite_start]e qual tem a maior média de preço (Amarilla)[cite: 18, 34].
2. [cite_start]**Análise Temporal:** Evolução mensal das vendas e lucros [cite: 49, 69][cite_start], com picos identificados no final do ano em outubro e dezembro[cite: 40, 43, 85, 89].
3. [cite_start]**Distribuição Geográfica:** Localização dos principais mercados na América do Norte e Europa (USA, Canadá, França, Alemanha e México)[cite: 100, 101, 102, 103].
4. [cite_start]**Lucratividade por Segmento:** Comparação de lucro entre setores como Small Business, Enterprise, Midmarket e Government[cite: 116, 117, 119, 120].

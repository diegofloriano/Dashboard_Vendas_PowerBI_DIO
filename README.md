# Dashboard de Vendas - Power BI (DIO)

Este é o meu primeiro projeto prático de Business Intelligence desenvolvido durante o curso da **DIO**. O objetivo foi transformar um dataset bruto de vendas em um relatório visual e interativo para facilitar a tomada de decisão.

## 📊 Visão Geral
O dashboard analisa o desempenho de vendas e lucro entre os anos de **2013 e 2014**, cobrindo diferentes produtos, segmentos e localizações geográficas.

### Principais Métricas (KPIs)
* **Vendas Totais:** 118,73 Mi
* **Produto Líder:** Paseo (29,35% do faturamento)
* **Segmento Mais Lucrativo:** Government (65,04% do lucro total)

## 📈 O que o Dashboard responde?
1. **Performance de Produtos:** Identifica qual produto traz mais receita e qual possui a maior média de preço.
2. **Análise Temporal:** Mostra a evolução mensal das vendas e lucros, com picos identificados no final do ano.
3. **Distribuição Geográfica:** Mapeia os principais mercados na América do Norte e Europa.
4. **Lucratividade por Segmento:** Compara o lucro entre setores como Small Business, Enterprise e Government.

## 🛠️ Tecnologias Utilizadas
* **Power BI** (Desktop)
* **DAX** (Cálculos de métricas e indicadores)
* **Dataset** (Dados de vendas globais)
